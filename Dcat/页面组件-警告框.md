警告框
Alert

```
<?php

use Dcat\Admin\Widgets\Alert;

$alert = Alert::make('内容', '标题');

// 类型
$alert->success();
$alert->danger();
$alert->info();
$alert->warning();

// 图标
$alert->icon('feather icon-x');

// 可移除按钮
$alert->removable();
```

Callout
```
<?php

use Dcat\Admin\Widgets\Callout;

$callout = Callout::make('内容', '标题');

// 颜色
$callout->light();
$callout->primary();

// 可移除按钮
$callout->removable();
```
